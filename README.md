suggest
=======

Find suggestions by entering pinyin abbreviations.

1. Run `psql < postgre.sql` to make database.
2. Get the `suggest` command line: `go get -v -u github.com/caiguanhao/suggest/suggest`

Requirements
------------

* PostgreSQL 9.5+
* Go 1.6+
